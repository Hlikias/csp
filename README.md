## 📖 项目简介

这是一个**模仿 QQ** 设计的局域网通信软件，主要用于 **Qt 学习**。它集合了 **C/S 架构**，既可以作为客户端，也可以作为服务器，适合在局域网中使用。目前实现了基础的文本通讯和图片传输功能。

> 练习用的，之前的完整版本找不到了，只有这个半残版本 😅 主要使用了 **XML**、**TCP**、**多线程** 等技术，数据库采用 **SQLite**。

后续计划将**串口**和**USB**相关功能从另一个项目融合过来，实现一个全功能的应用。有一定学习价值，会持续完善。

## ✨ 核心功能与实现

### 多线程并发
- **服务器端**：每个客户端连接由独立的工作线程处理，避免主界面卡顿，提升并发性能。
- **客户端**：收发消息、心跳检测等操作均在线程池中执行，保证界面流畅。

### 访问控制
- 基于 **SQLite** 存储用户信息，支持登录验证、好友关系管理。
- 服务器可限制特定 IP 或用户的连接权限。

### 黑名单机制
- 用户可屏蔽不想接收消息的对象，被屏蔽者的消息会被服务器直接丢弃。
- 黑名单列表同步保存在服务器数据库，多端登录时自动加载。

### 服务器运行状态检查
- 服务器端实时监控 CPU、内存占用，并在管理界面显示。
- 支持管理员手动刷新状态，查看在线用户数和连接数。

### 数据加密
- 登录密码使用 **SHA-256** 加盐哈希存储，确保数据库泄露时密码安全。
- 通信过程可选的 **AES-256** 加密（需手动启用），防止局域网嗅探。

### 心跳检查
- 客户端每隔 30 秒向服务器发送心跳包，服务器若连续三次未收到则判定客户端离线，清理会话并释放资源。
- 心跳机制也用于维持 NAT 穿透。

### 日志检索
- 服务器记录所有重要操作（登录、登出、异常断开等）至本地文件，支持按时间、用户、关键词检索。
- 客户端保留本地聊天记录，支持导出为 XML 或文本文件。

## 🛠️ 技术栈

- **框架**：Qt 5.15 (C++17)
- **网络**：QTcpServer / QTcpSocket，自定义协议基于 XML
- **数据库**：SQLite (用户信息、好友关系、黑名单)
- **并发**：QThreadPool + QRunnable，部分场景使用 QtConcurrent
- **加密**：QCryptographicHash (SHA-256)，第三方库实现 AES
- **日志**：自定义日志系统，支持滚动写入和按日期归档
- **配置**：XML 文件存储服务器设置、界面布局等
